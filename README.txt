KNPOS VERCEL YAYINLAMA

1. ZIP'i bilgisayarında ayıkla.
2. Vercel hesabına giriş yap.
3. Add New Project / Deploy bölümünü aç.
4. Ayıklanan klasörü yükle.
5. Framework: Other
6. Build Command: boş
7. Output Directory: boş
8. Deploy

Yayın sonunda verilen .vercel.app adresini telefondan aç.
iPhone: Safari > Paylaş > Ana Ekrana Ekle
Android: Chrome > Menü > Ana ekrana ekle
